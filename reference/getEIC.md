``` r
getEIC(

  Regime,
  TargetEvent,
  TargetTime,
  MinNuisance,

```

## Arguments

- Estimates:

  list

- Data:

  data.table

- Regime:

  list

- TargetEvent:

  numeric vector

- TargetTime:

  numeric vector

- MinNuisance:

  numeric

- GComp:

  boolean
