# getPropScore

getPropScore

## Usage

``` r
getPropScore(
  TrtVal,
  CovDT,
  TrtModel,
  MinNuisance,
  Regime,
  CVFolds,
  TrtLoss = NULL,
  ReturnModels
)
```

## Arguments

- TrtVal:

  numeric vector

- CovDT:

  data.table

- TrtModel:

  list or fitted object

- MinNuisance:

  numeric

- Regime:

  list

- CVFolds:

  list

- TrtLoss:

  character or function(A, g.A)

- ReturnModels:

  boolean
